# React Gen Image Components Plugin

A webpack plugin that scans an image directory and generates TypeScript React component files for each image.

## Installation

```bash
npm install @JamesB0010/portfolio-website-react-gen-image-components-plugin
```

## Usage

```ts
const ReactGenImageComponentsPlugin = require('@JamesB0010/portfolio-website-react-gen-image-components-plugin');
```
